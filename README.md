📱 Family App (Parents & Students)
An advanced educational management platform built with Flutter, designed to bridge the communication gap between educational institutions and families. The app features real-time tracking of student performance, attendance, and grades.

🌟 Key Features
Dual-Role Support: Dedicated interfaces for both Parents (Guardians) and Students.

Performance Tracking: Detailed visualization of exam results and academic progress.

Offline Support: Powered by HydratedBloc for viewing data without an internet connection.

Clean Architecture: Built using the BLoC/Cubit state management and GetIt for dependency injection.

🛠 Tech Stack
Framework: Flutter (Mobile & Web)

Language: Dart

State Management: Flutter BLoC & Cubit.

Local Storage: Hydrated BLoC & Path Provider.

Dependency Injection: GetIt.

Network: Dio with specialized Factory patterns.
